- 작성일시: 2026-05-08 07:02
- 범위: 주간현황, 월별가동현황, 대기/제안인원
- 목표: 기존 KPI 시트의 핵심 보고 화면을 브라우저에서 재현한다.

---

## 1. 주간현황

### 1.1 목표
`Ⅱ.1.주간현황`을 사람 수 기준 KPI 보고 화면으로 제공한다.

### 1.2 화면 구성
- 필터: 기준주차, 본부, 팀
- KPI: 현재 인원, 계약/수행 인원, 제안 인원, 비가동 인원, 가동률, 가득률
- 주간 요약 현황 테이블
- 대기 인원 현황(Snapshot)
- 제안 인원 현황(Snapshot)

### 1.3 요약 테이블 컬럼
- 구분
- 현재 인원
- 계약/수행 인원
- 제안 인원
- 비가동 인원
- 가동률
- 가득률

### 1.4 계산
- 사람 수 기준 가동률/가득률
- 전주 대비 증감 표시 가능

### 1.5 API 초안
- `GET /api/kpi/weekly`
- `POST /api/reports/weekly/export`

## 2. 월별가동현황

### 2.1 목표
`Ⅱ.2.월별가동현황`과 월별투입현황을 기반으로 월간 KPI 메인 화면을 제공한다.

### 2.2 화면 구성
- 필터: 연도, 기준월, 본부, 팀
- KPI: 평균 가동률, 평균 가득률, 수행 MM, 제안 MM, 지원 MM, 미투입 MM
- 가동률/가득률 추이 차트
- 월간 인력 투입 요약
- 월별 상세 현황 표

### 2.3 월별 상세 현황 표
- 구분
- 1월~12월
- 누계
- 전월 대비

### 2.4 계산
- 주말/공휴일 제외 MM 기준
- 월별 수행/제안/지원/미투입 MM 집계

### 2.5 API 초안
- `GET /api/kpi/monthly`
- `POST /api/reports/monthly/export`

## 3. 대기/제안인원

### 3.1 목표
`Ⅱ.3.대기,제안인원`을 기준일 기준 설명형 리스트 보고 화면으로 재현한다.

### 3.2 화면 구성
- 필터: 기준주차, 본부, 팀, 기준일
- KPI: 대기 인원, 제안/수행 인원, 합계
- 좌측: 대기 인원 리스트
- 우측: 제안/수행 인원 리스트
- 집계 기준 안내 문구

### 3.3 좌측 리스트 컬럼
- 본부
- 성명
- 직위
- 인원수
- 투입예정사업/비고

### 3.4 우측 리스트 컬럼
- 프로젝트명
- 성명
- 직위
- 투입종료
- 인원수

### 3.5 집계 규칙
- 본부/팀 필터는 인력 소속 기준 적용
- 우측 인원수는 프로젝트 묶음 기준 집계
- 화면 제목은 대기/제안인원, 우측 블록 명칭은 제안/수행 인원 사용

### 3.6 API 초안
- `GET /api/kpi/bench-proposal-staff`
- `POST /api/reports/bench-proposal-staff/export`
